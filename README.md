# AI Smart IT Helpdesk Platform

## Project Overview

AI Smart IT Helpdesk Platform is a web-based application developed using Python and Django. It allows users to register, log in, create IT support tickets, track complaint status, and receive AI-powered assistance.

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
- JavaScript
- Git
- GitHub

## Planned Features

- User Authentication
- Complaint Management (CRUD)
- Dashboard
- Admin Panel
- AI Complaint Summary
- AI Category Detection
- Search & Filter
- Role-Based Access
